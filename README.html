About

PreTax Calculator Pro is a web-based tool designed to help users calculate original prices before sales tax. Built with modern web technologies and featuring a dark theme for reduced eye strain.

Features

  • Reverse tax calculation
  • Step-by-step breakdown
  • Responsive design
  • Dark theme interface
  • No installation required

Usage Instructions

  1. Open index.html in any web browser
  2. Enter the total price (including tax)
  3. Input the tax rate percentage
  4. Click "Calculate" to see results

Development

This project uses vanilla HTML, CSS, and JavaScript for maximum compatibility and simplicity. No build process or dependencies required.
