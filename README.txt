PreTax Calculator Pro
====================

Description:
A professional-grade reverse sales tax calculator that helps users determine the original price of items before tax was applied. Built with a modern dark theme interface and responsive design.

Purpose:
This tool was created to simplify the process of calculating pre-tax prices from final totals, which is particularly useful for business accounting, expense tracking, and price comparison shopping.

Use Case Example:
A business expense manager finds a receipt showing $108.25 (tax included) but needs to record the pre-tax amount in their accounting software. With a local tax rate of 8.25%, they can quickly determine that the original price was $100.00.

How to Use:
1. Enter the final price (including tax)
2. Input the applicable tax rate percentage
3. Click "Calculate" to see the breakdown of:
   - Original price before tax
   - Tax amount
   - Total price with tax

Installation:
1. Clone the repository: git clone https://github.com/yourusername/pretax-calculator-pro
2. Open index.html in any modern web browser
3. No additional installation or dependencies required

Technical Requirements:
- Any modern web browser
- No server-side requirements
- Works offline
